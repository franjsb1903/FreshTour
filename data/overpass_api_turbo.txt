Para obter os datos dende OpenStreetMap, temos que empregar a ferramenta osm2pgsql (https://github.com/openstreetmap/osm2pgsql). Esta ferramenta necesita de arquivos OSM para cargar os datos na BBDD. Para iso, empregamos a API Overpass-Turbo (https://overpass-turbo.eu/#), que permite recuperar os datos en formato OSM, empregando as consultas abaixo especificadas. Simplemente temos que copiar e pegar a consulta na API e darlle a Executar. Unha vez cargue os datos, pulsamos sobre Exportar, e eliximos a opción de descargar en formato OSM (MOI IMPORTANTE). Seguidamente, situámonos no directorio da ferramenta osm2pgsql, e executamos os comandos correspondentes unha vez creada a BBDD e os esquemas empregados nelas (atención á localización dos aquivos OSM).


------------- LUGARES HOSPEDAXE -------------

/***** CONSULTA osm2pgsql *****/
.\osm2pgsql.exe -d fresh_tour --output-pgsql-schema=osm_lugares_hospedaxe -U postgres -P 5432 -S default.style -C 512 --hstore -H localhost -W -E 4326 ..\hospedaxe.osm
